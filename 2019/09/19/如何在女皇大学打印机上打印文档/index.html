如何在女皇大学图书馆打印文档

Posted on 2019-09-19 Edited on 2019-12-30 In 教程

据说明天就是due day了，你们一定在想怎么打印自己的文档吧！来来来，让我告诉你怎么操作，你们一定都可以交上作业的喔👊

教程内容：女皇学生如何给自己学生卡充打印费，如何使用打印机

需要准备的东西：

  1. 女皇学生的NetID
  2. 女皇学生卡
  3. 银行卡
  4. U盘*

以下是正文：

  1. 手机或电脑访问女皇图书馆网页（此处统一以电脑为例，手机操作和电脑一致，只是排版不一样）

  2. 点击右上角用户图标，然后再点击**Add Tri-Colour Cash** （见附图）

  3. 之后网页要求验证女皇NetID

  4. 验证成功之后，跳转进入账户页面，此次应该显示Tri-Colour金额为零。点击充值按钮（手机用户下拉至底部，电脑用户用户在右下角）
    记住我们充值的是Tri-Colour，不要充错啦！！！Flex是饭钱的！！！

  5. 输入金额，选择充值方法（左边是借记卡，右边是信用卡）

  6. 点击进入跳转到充值界面（以借记卡为例），左边是信用卡了，充值右边是在线银行的充值

  7. 点击充值，一会之后你的邮箱会收到支付凭证

P.S.

  1. Education Library 周一到周五早上都在8点钟开门
  2. 如果主页上金额还是没显示的话，需要等待2-3分钟，支付系统会有少许延迟。
  3. 详细开放时间和其他图书馆的位置信息请上女皇官网查询 https://library.queensu.ca/locations/all#all-locations-header

以Education Library为例，示范如何打印文件，其他女皇的图书馆大同小异，打印机找不到的话，可以去问图书管理员，他们会很乐意帮助你们滴

  1. 进入图书馆，图书馆有两个可供学生打印的打印机，分别在机房是门口和教师阅览室门口

  2. 你需要到机房下载文档（例如U盘，微信网页版，邮箱自己发给自己…)，然后点击打印。系统会需要你输入你的学生帐号和密码，第一次是用作开机，第二次是打印机打印验证。然后你再去打印机刷你的学生卡就可以打印了

  3. 打印完之后点击桌面上的关机图标，注销你的用户

    P.S.

    1. Education Library 周一到周五早上都在8点钟开门
    2. 如果主页上金额还是没显示的话，需要等待2-3分钟，支付系统会有少许延迟。
    3. 需要查询详细开放时间和其他图书馆的位置信息，请登录女皇官网查询: https://library.queensu.ca/locations/all#all-locations-header
    4. 打印机上U盘口是用来保存扫描的文件滴，是不可以打印的哦
    5. 如果碰巧机房在上课，可以去图书馆里面，任意一台有女皇标识的公用电脑都可以打印
    6. 如果同一台电脑需要打印不同人的文件，不需要切换账户，只需要在打印是输入那个人的NetID就可以了

    如有任何疑问, 欢迎在评论区留言
